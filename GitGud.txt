nano: edita o arquivo com o editor de texto "nano".

git init: torna a pasta um repositório git, criando o sistema de 
arquivos '.git/'

git add: adiciona um arquivo ao git, para que eles sejam commitados.

git commit: realiza um commit, salvando os dados no armazenamento 
remoto.

git branch: lista as branchs existentes.
git branch <argumento>: cria uma branch nova com o nome escrito.

git checkout <nome>: troca de branch.
git checkout <hash>: troca de versão. NOTA: qualquer arquivo commitado 
fica salvo no git, podendo ser restaurado. Não insira arquivos grandes.

.gitignore: Um arquivo responsável por avisar o git sobre quais arquivos 
NÃO sejam commitados.
